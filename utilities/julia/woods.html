wood(a,b,c) = (t) ->  a * t^b * ℯ^-(c*t)
w1 = wood(20,0.25,0.004) 
w2 = wood(30,0.15,0.004) 
w3 = wood(10,0.40,0.004)
plot([w1,w2, w3], 0,350, label = ["w1" "w2" "w3"])

In the plot above w1 has parameters similar to the overall means. The other two lactations appear to be similar except for a faster or slower rise after calving. Note that to achieve these shapes, I had to change both b and c parameters. These all look like credible lactations.

aYear = [-.48,-.66,-.33,-.028,.13,.30,.16,.054,.34,.64]

aVar = map(a -> wood(19.9+a, 26.89/100, 33.76/10000), bYear )
